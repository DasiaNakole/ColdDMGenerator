# Cold DM Generator for Freelancers

A polished outreach tool that helps freelancers turn their offer, proof, and target client into copy-ready cold DMs and follow-ups.

## Why This Is Marketable

Freelancers constantly need better outreach, and most of them waste time rewriting the same message over and over.

This project shows how to build a product that solves a real freelancer pain point:

- positioning offers clearly
- personalizing cold outreach
- generating multiple DM variants fast
- creating follow-ups without starting from scratch

## Features

- input form for service, target client, proof, offer, and CTA
- multiple message variants
- DM or email channel toggle
- copy buttons for each generated message
- polished responsive UI

## Tech Stack

- HTML
- CSS
- JavaScript
