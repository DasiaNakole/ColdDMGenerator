# Contra Listing Copy

## Title

Cold DM Generator for Freelancers

## Short Pitch

I build practical tools that help freelancers write better outreach faster, with less guesswork and more personalization.

## Description

This project is a cold outreach generator built specifically for freelancers who need to contact leads consistently without sounding robotic.

Users enter their service, target client, proof, offer, and call to action, and the tool generates multiple DM variants plus a follow-up message they can copy and use immediately.

## Deliverables

- polished frontend web app
- multiple generated outreach variants
- responsive interface
- copy-ready outputs
