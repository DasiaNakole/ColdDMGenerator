# Portfolio Case Study

## Title

Built a Cold DM Generator for Freelancers

## Summary

I built a lightweight outreach tool that helps freelancers generate sharper cold DMs and follow-ups from a few strategic inputs.

## Problem

Freelancers often know who they want to reach, but struggle to turn their offer into a short message that feels clear, personalized, and worth replying to.

## Solution

I created a generator that converts service details, target client information, proof, and CTA language into multiple outreach message variations.

## Outcome

The result is a practical portfolio app that demonstrates product thinking, UI design, and the ability to build tools around real freelancer workflows.
